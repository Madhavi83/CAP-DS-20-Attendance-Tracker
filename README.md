# CAP-DS-20: Classroom Attendance Tracker

## 📌 Project Description
This project implements a Classroom Attendance Tracker using Java and core Data Structures.

## 🧠 Data Structures Used
- HashMap → Student lookup (O(1))
- Queue → Daily attendance tracking
- ArrayList → Sorting attendance reports
- Comparator → Custom sorting logic

## 🚀 Features
- Add new students
- Mark attendance (Present/Absent)
- View student details
- View daily attendance order
- Generate sorted attendance report
- Start new day (clear daily queue)
- Input validation

## 🛠 Technologies Used
- Java
- OOP Concepts
- Collections Framework

## ▶ How to Run
```bash
cd src
javac *.java
java Main
```


## 📊 Complexity Analysis

- Student Lookup → O(1)
- Mark Attendance → O(1)
- Report Generation → O(n log n)
